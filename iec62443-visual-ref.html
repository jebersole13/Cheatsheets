Terminology Quick Reference (Confusing Terms Decoded)

SL-T (Target)
What security level you WANT to achieve based on risk
SL-A (Achieved)
What security level you've ACTUALLY implemented
SL-C (Capability)
What security level a PRODUCT can support
FR (Foundational Req)
7 high-level categories (like chapters)
SR (System Req)
Specific technical controls under each FR
RE (Req Enhancement)
Stronger versions of SR for higher security levels
CR (Component Req)
Like SR but for products (Part 4-2)
Zone
Group of assets with same security level
Conduit
Controlled connection BETWEEN zones
